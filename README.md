# MUNDO.GRID

Características principales:

Diseño Premium: Fondo oscuro con gradientes radiales, tipografía moderna ('Outfit') y tarjetas con efectos de cristal y resplandor.
Interactividad sin JS: He utilizado el "CSS Checkbox Hack" para permitir que al hacer clic en un país se muestre su capital (y viceversa) de forma fluida, cumpliendo con tu restricción de no usar otros lenguajes.
Listo para GitHub Pages: He generado un archivo index.html estático en la raíz que puedes subir directamente a GitHub.
Python Backend: Incluye un servidor Flask para desarrollo local y un script de exportación para generar la versión estática.
